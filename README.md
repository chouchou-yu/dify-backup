# dify-backup
# 基于本地大模型的个人知识问答助手

## 项目简介
本项目基于 Dify 平台和 Ollama 本地大模型，构建了一个个人知识问答助手，
支持通过自定义知识库进行检索增强生成，实现安全、可控的本地问答。

## 技术栈
- Ollama
- Qwen3:8B
- Dify（本地部署）

## 系统架构
用户问题 → 向量化 → 知识库检索 → 本地大模型生成回答

## 功能演示
- 知识库问答
- 无命中信息拒答
- 本地运行，无数据外泄

## 知识库构建
- 文档格式：Markdown
- 分段策略：Chunk Size 200 / Overlap 50
- 检索参数：Top K = 3
